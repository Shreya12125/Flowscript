Getting Started

FlowScript uses natural language syntax:

  • Variables: Create a number called x with value 10
  • Functions: Define function add with parameters a and b
  • Loops: while condition: do something
  • Conditions: if condition then do this

Basic Syntax

Variables:
Create a number called age with value 25
Create a text called name with value "Alice"
Create a boolean called isStudent with value true
Functions:
Define function greet with parameter name:
    say "Hello " plus name
Loops:
while counter < 10:
    say counter
    increase counter by 1
